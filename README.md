# Placement Readiness Platform

A premium, offline-first placement preparation workspace with JD analysis, readiness scoring, history, and export tools.

## Features
- Landing + dashboard with readiness overview and progress widgets
- JD analyzer with skill extraction, round mapping, and 7-day plan
- Interactive results with skill confidence toggles and live score updates
- History with localStorage persistence and safe schema normalization
- Export tools (copy sections, download TXT)
- Test checklist and ship gate

## Tech Stack
- React + Vite
- Tailwind CSS
- React Router
- Recharts
